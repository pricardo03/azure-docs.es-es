A partir del 1 de julio de 2018, se elimina la compatibilidad con TLS 1.0 y 1.1 de Azure VPN Gateway. VPN Gateway será solo compatible con TLS 1.2. Esto solo afecta a las conexiones de punto a a sitio, las conexiones de sitio a sitio no se verán afectadas. Si usa TLS para VPN de punto a sitio en clientes de Windows 10, no necesita hacer nada. Si usa TLS para conexiones de punto a sitio en clientes de Windows 7 y Windows 8, consulte las [preguntas más frecuentes sobre VPN Gateway](../articles/vpn-gateway/vpn-gateway-vpn-faq.md#P2S) para obtener instrucciones de actualización.
