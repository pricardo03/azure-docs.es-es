En la parte superior de la clase Program, agregue las siguientes constantes para acceder a QnA Maker:

```csharp
// Represents the various elements used to create HTTP request URIs
// for QnA Maker operations.
static string host = "https://westus.api.cognitive.microsoft.com";
static string service = "/qnamaker/v4.0";
static string method = "/knowledgebases/create";

// NOTE: Replace this value with a valid QnA Maker subscription key.
static string key = "<your-qna-maker-subscription-key>";
```

Las tres primeras constantes se usan para crear la dirección URL completa de la API. La última constante proporciona autenticación en la API.
