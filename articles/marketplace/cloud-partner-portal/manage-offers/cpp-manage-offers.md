# <a name="manage-azure-and-appsource-marketplace-offers"></a>Administración de las ofertas del Marketplace de Azure y AppSource

En esta sección se presentan las operaciones generales que se utilizan para administrar las ofertas del Marketplace de Azure y AppSource.  Por ejemplo, se explica cómo utilizar [Cloud Partner Portal](https://cloudpartner.azure.com/) para crear y publicar una oferta de Marketplace.  En esta sección también se explican problemas técnicos que son comunes en la mayoría de los tipos de ofertas, como los recursos basados en texto y de imagen.


## <a name="standard-offer-operations"></a>Operaciones estándar de las ofertas

Cloud Partner Portal permite que los anunciantes realicen las siguientes operaciones básicas en una oferta.

|     Operación      |  DESCRIPCIÓN                                           |
|     ---------      |  -----------                                           |
| [Creación de oferta](./cpp-create-offer.md)   | Crea una entrada de oferta pendiente en el repositorio de incorporación, pero no la publica automáticamente en un Marketplace de Microsoft. | 
| [Visualización del estado de la oferta](./cpp-view-status-offer.md)   | Proporciona el estado de publicación de una oferta.  En el portal existen varios lugares en los que se muestra la información de estado. |
| [Publicación de oferta](./cpp-publish-offer.md) | Valida y, a continuación, envía una entrada de oferta completa al Marketplace de Microsoft adecuado.  A continuación, se dice que la oferta está *publicada*. |
| [Actualización de la oferta](./cpp-update-offer.md)   | Permite modificar ciertas características en una oferta existente.  Se aplica a las ofertas publicadas y pendientes. |
| [Eliminación de la oferta](./cpp-delete-offer.md)   | Elimina una oferta pendiente o publicada existente.  | 
|  |  |
