# <a name="dynamics-365-for-operations-help-wiki"></a>Wiki de ayuda de Dynamics 365 for Operations

Microsoft Dynamics 365 for Operations es la aplicación empresarial de Microsoft para las grandes empresas. Permite que los usuarios tomen rápidamente decisiones más inteligentes a través de una interfaz de usuario inteligente que brinda acceso a información y datos en tiempo real. Permite que los negocios se transformen e innoven al brindar una lógica de negocios comprobada que les permite volver a diseñar más rápido sus procesos de negocio. Ofrece a los negocios la flexibilidad para crecer a su propio ritmo y hacer negocios prácticamente en cualquier parte, en cualquier lugar, lo que les permite escalar globalmente sus operaciones para satisfacer las necesidades de negocio.

Esta guía de usuario puede ayudarlo a obtener información sobre cómo hacer que Dynamics 365 for Operations funcione para su negocio.

Consulte la [wiki de ayuda de Dynamics 365 for Operations](https://docs.microsoft.com/dynamics365/operations/) para más información.
