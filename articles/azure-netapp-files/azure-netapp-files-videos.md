# <a name="azure-netapp-files-videos"></a>Vídeos de Azure NetApp Files
Este artículo proporciona referencias a vídeos que contienen discusiones detalladas sobre el uso de Azure NetApp Files. 

Hay disponibles varios vídeos para ayudarle a obtener información sobre Azure NetApp Files:  

* [Microsoft Ignite 2019: Ejecución de las cargas de trabajo de archivos empresariales más exigentes con Azure NetApp Files](https://myignite.techcommunity.microsoft.com/sessions/82938?source=sessions) proporciona una breve introducción a Azure NetApp Files, incluidos los casos de uso y una demostración y luego profundiza en las funcionalidades y el mapa de ruta.
* [Kirk Ryan habla sobre Azure NetApp Files](https://www.youtube.com/channel/UCq1jZkyVXqMsMSIvScBE2qg/playlists) es una serie de vídeos, tutoriales y demostraciones dedicados a Azure NetApp Files.
