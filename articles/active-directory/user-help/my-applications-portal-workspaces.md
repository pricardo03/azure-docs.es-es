# <a name="access-and-use-workspaces-on-the-my-apps-preview-portal"></a>Acceso y uso de las áreas de trabajo en el portal Aplicaciones (versión preliminar)

Las áreas de trabajo son diferentes vistas de las aplicaciones que se muestran en el portal **Aplicaciones** (versión preliminar). Las aplicaciones se pueden agrupar en áreas de trabajo por función de departamento o rol de usuario, por ejemplo. Si no ve ningún área de trabajo o categoría disponible, significa que el administrador no ha configurado ni compartido ninguna con usted. Póngase en contacto con el departamento de soporte técnico de la organización para obtener ayuda adicional o permisos para ver las áreas de trabajo compartidas.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-and-workspaces.md)]

>[!Important]
>Este contenido está diseñado para usuarios. Si es administrador, puede encontrar más información acerca de cómo configurar y administrar aplicaciones basadas en la nube en la [Documentación sobre la administración de aplicaciones](https://docs.microsoft.com/azure/active-directory/manage-apps/access-panel-workspaces).

## <a name="access-apps-using-workspaces"></a>Acceso a las aplicaciones mediante las áreas de trabajo

De forma predeterminada, la lista de áreas de trabajo del portal **Aplicaciones** muestra una lista denominada **Todas las aplicaciones**, que incluye todas las aplicaciones a las que tiene acceso:

![Página Todas las aplicaciones del portal Aplicaciones](media/my-applications-portal-workspaces/my-apps-all.png)

Cualquier otra área de trabajo que vea en esta lista la creó el administrador y la compartió con usted. Puede elegir una de esas áreas de trabajo para ver un conjunto de aplicaciones más amplio, como *Compatibilidad* en el ejemplo siguiente:

![Área de trabajo individual y conjunto de aplicaciones con ámbito en el portal Aplicaciones](media/my-applications-portal-workspaces/my-apps-workspace.png)

Al seleccionar y ver un área de trabajo, solo verá las aplicaciones predeterminadas a las que tiene acceso.

## <a name="next-steps"></a>Pasos siguientes

Después de organizar las aplicaciones en las distintas categorías del portal **Aplicaciones**, puede:

- Revisar, actualizar o revocar los permisos concedidos a las aplicaciones. Para más información, vea [Edición o revocación de permisos de aplicación en el portal Aplicaciones (versión preliminar)](my-applications-portal-permissions-saved-accounts.md).
