# <a name="how-to-recognize-differences-between-delegated-and-application-permissions"></a>Reconocimiento de las diferencias entre permisos delegados y permisos de aplicación

## <a name="recommended-documents"></a>Documentos recomendados

- Descubra cómo usan las aplicaciones cliente las [solicitudes de permisos delegados y de aplicación](https://docs.microsoft.com/azure/active-directory/develop/active-directory-dev-glossary#permissions) para acceder a los recursos.
- Obtenga instrucciones paso a paso acerca de cómo [configurar las solicitudes de permiso de una aplicación cliente](https://docs.microsoft.com/azure/active-directory/develop/active-directory-integrating-applications#configuring-a-client-application-to-access-web-apis).
- Para más detalles, infórmese acerca de cómo las aplicaciones de recursos exponen [ámbitos](https://docs.microsoft.com/azure/active-directory/develop/active-directory-dev-glossary#scopes) y [roles de aplicación](https://docs.microsoft.com/azure/active-directory/develop/active-directory-dev-glossary#roles) a las aplicaciones cliente, que se manifiestan en Azure Portal como permisos delegados y de aplicación, respectivamente.
