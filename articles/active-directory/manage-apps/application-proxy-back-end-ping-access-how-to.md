# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>Configuración de una aplicación de proxy de aplicación para usar PingAccess

Nuestra colaboración con PingAccess ahora nos permite extender las ventajas del proxy de aplicación a aquellas aplicaciones que usen la autenticación basada en el encabezado. Si las aplicaciones no utilizan encabezados, consulte nuestra [documentación sobre el inicio de sesión único](application-proxy-configure-single-sign-on-with-kcd.md) para ver detalles sobre otras opciones.

## <a name="overview-of-steps-and-recommended-documents"></a>Introducción a los pasos y documentos recomendados

Para configurar una aplicación con PingAccess, hay cuatro pasos:

1.  Configurar los conectores del proxy de aplicación

2.  Crear una aplicación de proxy de aplicación de Azure AD

3.  Descargar y configurar PingAccess

4.  Configurar aplicaciones en PingAccess

Para ver información detallada sobre cada uno de estos pasos, consulte nuestra [documentación sobre el inicio de sesión único con encabezados](application-proxy-configure-single-sign-on-with-ping-access.md).
