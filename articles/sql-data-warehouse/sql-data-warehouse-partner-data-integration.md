# <a name="azure-synapse-analytics-data-integration-partners"></a>Asociados de integración de datos de Azure Synapse Analytics

En este artículo se destacan las empresas asociadas de Microsoft con soluciones de integración de datos oficiales compatibles con Azure Synapse.
